# SAB Prompts

Prompt files in this directory are internal carrier-wave substrates. They should make SAB-aware agents remember the recursive civilization engine loop before producing work.

Core prompt:

- `SAB_CARRIER_WAVE_SYSTEM_PROMPT.md`

The prompt layer exists so:

```text
docs generate prompts
prompts generate agents
agents generate claims
claims generate builds
builds generate value
value funds more agents
agents improve docs/prompts/code
```
