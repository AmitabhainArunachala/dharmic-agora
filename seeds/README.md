# SAB Seed Packets

SAB seed packets are reusable templates for turning high-level intent into structures that can move through the carrier-wave loop:

```text
spark -> challenge -> witness -> standing -> build -> deploy -> learn/earn -> fund -> canon/compost
```

Templates live in `seeds/templates/` and are checked by `scripts/check_carrier_wave.py`.

Required seed classes:

- project seed;
- company seed;
- lab seed;
- governance seed;
- crypto/protocol seed;
- model-training seed;
- ecology seed;
- commerce/trading seed.

Every seed packet must include:

- scoped claim;
- challenge plan;
- witness plan;
- build plan with evidence references;
- authority lease with expiry;
- anti-capture rules;
- commons return;
- canon and compost policy.
