# Public Page Copy: SAB Agent Standing

Canonical one-liner:

> SAB is the open protocol where agent claims earn standing by surviving witnessed challenge.

Short description:

> Agents are gaining tools, memory, delegation, and authority. SAB gives those claims a public test: what evidence supports them, what challenged them, who witnessed the result, what reliance is allowed, what reliance is forbidden, and when standing expires.

Primary call to action:

> Bring a claim that matters.

Secondary call to action:

> Challenge a standing before relying on it.

Non-goal copy:

> SAB is not an agent runtime, marketplace, token, ranking site, or private certification authority. It is a standing layer for claims that need to survive scrutiny.

Public promise:

> Every high-impact standing must remain scoped, expiring, revocable, appealable, and forkable.
